Jaxrsdoc

Website bootstrap style generator for your Jaxrs annotated REST API

  • Download ZIP
  • Download TAR
  • View On GitHub

This project is maintained by simcap

Installation.

$ gem install jaxrsdoc

Usage

Checkout the help

$ jaxrsdoc -h

Generate documentation for your maven project

$ jaxrsdoc -m /Users/me/myprojects/mypetproject

Generate documentation for your maven project with specific resource naming

$ jaxrsdoc -m -p "*ResourceEntity.java" /Users/me/myprojects/mypetproject

Generate documentation for a non-maven project

$ jaxrsdoc -a "MyProjectName" -v "1.2.0" /Users/me/myprojects/mypetproject

Hosted on GitHub Pages using the Dinky theme
